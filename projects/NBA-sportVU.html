NBA SportVU animations

Here are some examples of animations of tracking data from an NBA game. The game was a Golden State Warriors - Los Angeles Lakers from January 2016.
Step 1 - Data collection
I downloaded the data from linouk23's github and extracted it as JSON.
Step 2 - Data manipulation
I got started following Dan Vatterorot's post, which was very helpful to understand the structure of the data and the correct approach.
Step 3 - Data visualisation
After using matplotlib's FuncAnimation function (took me a while to understand how it works), I used an image of the court found in Savvas Tjortjoglou's post, also very helpful. The final step was to save the animations as htlm5 videos.
